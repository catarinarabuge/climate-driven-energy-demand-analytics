Ingestão de Dados

Esta operação irá correr a ingestão completa — ENTSO-E (carga eléctrica) e ERA5 (meteorologia). Pode demorar vários minutos.
Correr run_ingestion_raw() — ENTSO-E
Correr download_era5() → ingest_era5()

Cleaning e Alinhamento

Corre o cleaning de energia e une os datasets num único ficheiro merged.csv.
Correr run_processing()
Correr merge_datasets(...) → merged.csv

Feature Engineering

Gera as features base (v1) a partir do merged.csv e depois as features avançadas (v2).
generate_features() → features_v1.csv
generate_advanced_features() → features_v2_advanced.csv

Treinar Modelos

Requer confirmação de credenciais. Treina Linear Regression e Random Forest usando features_v2_advanced.csv.

Métricas e Resultados
